# How to use Jupyter4NFDI?
This is a self-paced training for anyone who wants to learn how to use Jupyter4NFDI for their research.  You will learn  how to create
Python notebooks, R notebooks, or other-format work on
[Jupyter4NFDI](https://nfdi-jupyter.de/), publish it  via GitHub, make it reproducible using `repo2docker`, and validate it automatically with GitHub Actions.
